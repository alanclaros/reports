TeamWork - Sistema de Administración y Control de Condominios

 
 

El objetivo de este sistema es el control y administración de departamentos dentro del condominio, realizando cobros por consumo de agua, gastos de expensas mensuales, gastos de mantenimiento, multas por retrasos y todos los gastos requeridos por el condominio. Contando además con la impresión de recibos, reportes, listado de cobros, etc.

También facilitará la reserva de espacios comunes a través del sitio web realizando las reservas mediante cualquier dispositivo móvil, Tablet, PC.

Para obtener má detalles de todos los módulos y benificios de este sistema, dirigase a la página de Contacto 



 

Módulo de Cobros
El sistema realizará cobros por cada uno de los departamentos dentro el condominio. Se generará un recibo en formato PDF, que puede ser enviado al dueño del departamento o realizar la impresi&oan del mismo; por cada periodo de cobro.

Se mostrará una ventana de búsqueda de los departamentos, con las opciones de bloque, piso, dueño del departamento, etc. Al seleccionar cada departamento se mostrarán los detalles de deudas del departamento por cada periodo.

Realizando las operaciones de cobro por cada periodo y la impresión de recibos por cada periodo.


 

Reserva de Áreas Comunes
El sistema mostrará un calendario con todas las áreas comunes del condominio, donde podrán realizarse la reserva de estos espacios, solicitando al administrador esta reserva indicando la fecha y el perido de tiempo que desea reservar una determinada área.

Todos los usuarios del sistema podrán ver este calendario para poder ver y planificar su reserva de una forma má comoda, teniendo a la vista la informació de todas las reservas ya confirmadas.


 

Impresión de Recibos
El sistema mostrará un recibo por cada periodo de Cobro en formato PDF, mostrando el detalle de la deuda por cada periodo y para cada departamento.

Este recibo puede ser enviado a cada departamento del condominio ya sea via correo electrónico, WhatsApp, o mediante cualquier medio que disponga, como también puede realizar la impresión del mismo.

EV-Control - Sistema de Administración y Control de Eventos Sociales

 
 

El objetivo de este sistema es el control y administración del Alquiler de Artículos para Eventos Sociales, realizando el control de stock para cada nuevo evento, tomando en cuenta la salida de artículos por el contrato de otros eventos, como también la vuelta de artículos a almacén de otros contratos.

Contando además con la impresión de proformas, recibo de ventas, salida de productos de almacén, retorno de productos a almacén con sus respectivos costos adicionales por rotura y/o refacción, y una liquidación de toda la deuda acumulada para el cliente.

Para obtener má detalles de todos los módulos y benificios de este sistema, dirigase a la página de Contacto 




 

Módulo de Ventas
El sistema realizará el control de todos los contratos realizados por la empresa, registrando:

  • Preventas, proforma de artículos para los clientes
  • Ventas, confirmación del contrato
  • Salida de Almacén, Registro de la salida de los artículos de almacén
  • Vuelta a Almacén, retorno de los artículos a almacén
  • Finalización, cancelación de la deuda total por parte del cliente

Se mostrarán notificaciones de aviso en el Celular o PC, con anticipación, de todos los contratos que deben ser entregados o recogidos del evento social.


 

Impresión de Recibos
El sistema mostrará un archivo PDF de impresión, dependiendo el estado de la venta :

  • Preventa, proforma para el cliente
  • Venta, impresión del contrato con el cliente
  • Salida, indicando la cantidad de artículos que salen de almacén
  • Vuelta, indicanto la cantidad de artículos que vuelven a almacén, además de registrar costos de rotura y refacción
  • Finalización, deuda total del cliente

Al-Instante - Sistema de Administración y Control de Restaurantes

 
 

El objetivo de este sistema es el control y administración de los productos ofrecidos por el resturante, dando la posibilidad de configurar cada producto, con sus respectivos aderezos y acompañantes, configurando también opciones de combos en cada producto, como ser :

  • Papas
  • Refrescos
  • O Cualquier otro que requiera

Cuenta también con un control de stock de cada producto, dependiendo las configuraciones que tenga, descontando 1 o varios elementos en cada venta de un producto.

Para obtener má detalles de todos los módulos y benificios de este sistema, dirigase a la página de Contacto 



 

Módulo de Pedidos
El sistema mostrará el detalle de las configuraciones de cada producto, dando la opción al usuario de elegir los ingredientes y acompañantes para cada producto.


 

En caso de combos que tengan papas, refrescos, etc. Se mostrarán las opciones de agrandar cada un de estos elementos aumentando o disminuyendo el precio del producto.


 

Se mostrarán ventanas de confirmación para los ingredientes y extras de cada producto, dando la opción de quitar algunos o aumantar otros con su respectivo costo adicional si es el caso.
