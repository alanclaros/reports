## Create an virtual environment

```shell
python -m venv venv
```

## Activate virtual environment

```shell
.\venv\Scripts\activate
```

## Install dependencies

```shell
pip install -r requirements.txt
```

## Create a new lambda.

```shell
apm lambda new
```

## Create a new lambda with apigateway integration.

```shell
apm endpoint new
```
